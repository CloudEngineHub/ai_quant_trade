# 评级体系说明

券商研报评级口径不统一，东方财富做了标准化映射。本 Skill 以 `emRatingName`（标准化评级）为主要分析字段，`sRatingName`（机构原始评级）作辅助。

## 东方财富标准化评级

| 标准评级 | 含义 | 方向 |
|---|---|---|
| 买入 | 预期未来 12 个月跑赢大盘 15%+ | 看多 |
| 增持 | 预期跑赢 5%-15% | 偏多 |
| 中性 | 预期跑赢 -5%-5% | 中性 |
| 减持 | 预期跑输 5%-15% | 偏空 |
| 卖出 | 预期跑输 15%+ | 看空 |

## 机构原始评级映射（常见）

| 机构原始评级 | 对应标准化 | 说明 |
|---|---|---|
| 强烈推荐 / 优于大市 / 推荐 | 买入 ~ 增持 | 各家口径略有差异 |
| 买入 / 增持 / 谨慎推荐 | 买入 ~ 增持 | |
| 审慎推荐 / 推荐 | 增持 | |
| 中性 / 同步大市 / 回避 | 中性 ~ 减持 | |
| 减持 / 卖出 | 减持 ~ 卖出 | |

## 评级变动（ratingChange 字段）

| 代码 | 含义 |
|---|---|
| 1 | 首次评级 |
| 2 | 上调 |
| 3 | 维持 |
| 4 | 下调 |
| 5 | 撤销 |

## 分析注意事项

- 评级方向比评级绝对值更重要（维持买入 vs 上调至买入，信号强度不同）
- "首次评级"参考价值有限，缺乏历史连贯性
- 评级变动（尤其下调）比静态评级更有信息量
- 卖方研报普遍偏多，"中性"往往已是偏谨慎信号
