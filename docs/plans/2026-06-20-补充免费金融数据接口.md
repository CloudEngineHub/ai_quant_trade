# 补充免费金融数据接口实施计划

> **目标：** 在 `egs_data/` 下补充各类**免费**金融数据接口的案例，覆盖股票、期货、加密货币、宏观经济、官方披露等多个维度，与现有 baostock/qstock/tushare/web_api/yfinance 形成完整的免费数据获取示例集。

**架构：** 每个数据源一个独立子目录，包含 `README.md`（安装+接口说明+示例）和可运行的示例脚本（`.py` 或 `.ipynb`）。遵循现有目录的 Apache 2.0 协议头、中文文档风格。

**技术栈：** Python 3、pandas、requests、akshare、efinance、qlib、ccxt、fredapi、pandas-datareader
