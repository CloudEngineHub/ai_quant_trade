# 1. 简介

网易财经提供免费的历史数据下载接口，支持 CSV 格式下载，数据较为完整。
适合获取股票历史行情和财务数据。

优点：
* 完全免费，无需注册
* 支持 CSV 直接下载，数据格式规整
* 历史数据较长，可追溯至上市首日

缺点：
* 无官方 API 文档
* 实时行情接口不稳定
* 部分接口已停止维护

# 2. 接口域名

| 域名 | 用途 |
|------|------|
| quotes.money.163.com | 历史数据下载（CSV） |
| api.money.126.net | 实时行情（JSON） |

# 3. 代码格式

网易财经的股票代码格式与市场有关：
| 市场 | 格式 | 示例 |
|------|------|------|
| 沪市 | 0 + 6位代码 | 0601318 |
| 深市 | 1 + 6位代码 | 1000001 |

# 4. 字段说明

历史数据下载接口支持的字段：
| 字段 | 说明 |
|------|------|
| TCLOSE | 收盘价 |
| HIGH | 最高价 |
| LOW | 最低价 |
| TOPEN | 开盘价 |
| LCLOSE | 前收盘价 |
| CHG | 涨跌额 |
| PCHG | 涨跌幅 |
| TURNOVER | 换手率 |
| VOTURNOVER | 成交量 |
| VATURNOVER | 成交额 |
| TCAP | 总市值 |
| MCAP | 流通市值 |

# 5. 注意事项
* 沪市代码前缀为 0，深市代码前缀为 1
* 历史数据接口返回 CSV，可直接用 pandas 读取
* 部分老接口可能已失效，以实际测试为准
