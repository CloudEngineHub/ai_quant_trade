# 1. 简介

东方财富网提供免费的金融数据接口，无需注册，直接通过 HTTP 请求获取 JSON 数据。
数据覆盖股票、指数、基金、债券等，是 akshare、efinance 等库的底层数据源之一。

优点：
* 完全免费，无需注册和 API Key
* 数据丰富，覆盖面广
* 返回 JSON 格式，解析方便

缺点：
* 无官方 API 文档，字段编码需自行摸索
* 接口可能随网站更新而变化
* 无频率保证，高频调用可能被限制

# 2. 接口域名

| 域名 | 用途 |
|------|------|
| push2his.eastmoney.com | 历史K线数据 |
| push2.eastmoney.com | 实时行情数据 |
| push2his.eastmoney.com/api/qt/stock/fflow | 资金流向数据 |
| datacenter-web.eastmoney.com | 数据中心（财务、龙虎榜等） |

# 3. 字段编码说明

## secid（证券ID）
格式：`市场.代码`
- `1.601318`：沪市股票（1=沪市）
- `0.000001`：深市股票（0=深市）

## klt（K线类型）
| 值 | 说明 |
|----|------|
| 5 | 5分钟 |
| 15 | 15分钟 |
| 30 | 30分钟 |
| 60 | 60分钟 |
| 101 | 日K |
| 102 | 周K |
| 103 | 月K |

## fqt（复权类型）
| 值 | 说明 |
|----|------|
| 0 | 不复权 |
| 1 | 前复权 |
| 2 | 后复权 |

## f字段（行情数据）
| 字段 | 说明 |
|------|------|
| f51 | 日期 |
| f52 | 开盘 |
| f53 | 收盘 |
| f54 | 最高 |
| f55 | 最低 |
| f56 | 成交量 |
| f57 | 成交额 |
| f58 | 振幅 |
| f59 | 涨跌幅 |
| f60 | 涨跌额 |
| f61 | 换手率 |

# 4. 注意事项
* 无官方文档，字段含义可能变化
* 建议调用间隔大于 0.5 秒
* 部分接口需要设置 Referer 头
