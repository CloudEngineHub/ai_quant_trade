# 1. 简介

腾讯财经提供免费的股票数据接口，无需注册，返回文本或 JSON 格式。
数据覆盖实时行情、历史K线、分钟数据等。

优点：
* 完全免费，无需注册
* 接口稳定，响应速度快
* 支持实时行情和历史K线

缺点：
* 无官方 API 文档
* 历史数据量有限
* 返回格式不统一，需自行解析

# 2. 接口域名

| 域名 | 用途 |
|------|------|
| qt.gtimg.cn | 实时行情（文本格式） |
| web.ifzq.gtimg.cn | 历史日K线（JSON） |
| ifzq.gtimg.cn | 分钟K线（JSON） |

# 3. 代码格式

| 市场 | 格式 | 示例 |
|------|------|------|
| 沪市股票 | sh + 代码 | sh601318 |
| 深市股票 | sz + 代码 | sz000001 |
| 沪市指数 | sh + 代码 | sh000001 |
| 深市指数 | sz + 代码 | sz399001 |
| 港股 | hk + 代码 | hk00700 |
| 美股 | us + 代码 | usAAPL |

# 4. 注意事项
* 实时行情接口返回文本格式，需自行分割解析
* 历史K线最多返回 640 根
* 调用频率建议间隔大于 0.5 秒
